For general token auth info:

GETTING A TOKEN (need httpie installed):
Note: to do with curl, have to explicitly use JSON; easier with httpie
http POST 127.0.0.1:8000/api-token-auth/ username=jarbaughtest password=jarbaughtest

THEORETICALLY ACCESSING A PAGE REQUIRING AUTH
curl -X GET http://127.0.0.1:8000/auth/ -H 'Authorization: Token 2b522cb500a9a2c17b0b973ec5b053c5a4580325'
(the above token corresponds to the superuser jarbaughtest)
